== Description ==

Display a live REALTIME twitter view of a chosen keyword


== Installation ==


1. Upload the folder to your `/wp-content/plugins/` directory

2. Activate the plugin

3. To edit the settings go to your widget selctions panel `wp-admin/widgets.php` 
	and select zMonitter
